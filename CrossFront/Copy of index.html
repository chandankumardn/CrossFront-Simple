What is CrossFront
CrossFront is a boilerplate crossplatform frontend framework to help you quickly build desktop/tablet/mobile applications using the motto "code once - run everywhere". If you have ever tried to develop a mobile/desktop application using jquery mobile and phonegap it works great for simple ideas and proof of concept sites. When you graduate to developing a full blown application the code becomes a nightmare of tangled spaghetti code and framework sprawl. Unless you are a javascript master and maintain your own way to manage and organize code you need to use a combination of the many javascript frameworks out there. This kickstart project will provide you with a crossplatform framework along with tutorial videos on how create an application from scratch. Server side code is not in scope for this project and assumes a RESTful web api backend.

I started this project when attempting to build a Facebook Mobile/Desktop applcation. I quickly realized I sucked at javascript and needed the right recipe of frameworks to make this project do-able. After A ton of searching I relized nobody has really put all the frameworks together in a nice boilerplate. In the process I learned how to do the AMD approach. This basically reminds me of traditional object oriented programming languages with good organization and structure. This is my attempt at solving this problem.

Frameworks used:
  • jquery - DOM manipulation
  • jquery mobile - Mobile UI and touch events
  • jquery mobile router - Hash routing
  • Backbone - Code organization
  • Underscore - For utilities
  • Require.js - For module loading and namespacing
  • phonegap - For crossplatform functionality. iOS, Android etc... Also used for saving to local database when offline, then sync to database
  • Offline vs. Online Only support
  • TypeScript - Code structure and organization (classes, modules, intellisense, AMD), coding is similar to c#

Key Takeaways:
This project will show you how to build a base boilerplate starting point for your application. It will enable you to quickly build scalable iPhone, Android, Windows Phone, Tablet, and Desktop applications.
